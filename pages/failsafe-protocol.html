Failsafe Protocol

About

Failsafe Protocol was made for the Bigmode Game Jame 2025 with the theme 'POWER'. I made the game with my brother where he worked on a few pixel arts. We hope to someday continue developing this game, add new features and polish it further.

Project Info

Team Size: 2

Time Frame: 1 week

Engine: LÖVE2D

Introduction

This resource management like game was a fun challenge for me to get into the world of game jams as it was the first game I made for a game jam. I had learned Lua for LÖVE2D recently as well so I wanted to refine my skills with lua further by making another game with it.

Gameplay GIF's from Failsafe protocol.

What I learned

The main goal for this project was to make a full game loop in a game framework that I recently learned how to use. This project was a great way to reach my goal. I also was curious about how games are made inside game frameworks and this project was a great way of learning how.

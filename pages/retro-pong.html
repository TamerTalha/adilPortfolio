About

Retro Pong was my first project that I 'finished' and distributed for the first time. It was initially just the product of a lecture from CS50's Introduction to Game Development. But after making the assignment for the lecture, it led me to want to work on it more.

Introduction

As it says in the description of the game in my itch.io page, this is a simple pong game with some cool shaders in my opinion. After making the assignment for the lecture I made more difficulties using different methods and then I proceeded to add some shaders that I found on internet.

Gameplay GIF's from Retro Pong.

What I learned

The main goal of this project was to just make something finished and distribute it somewhere. Which is achieved. Initially putting shaders was not the plan but after discovering the world of shaders I got very interested in them and decided to put in a few in Retro Pong. I especially had a nice challenge with the shockwave shader that plays when it is scored. With the help of this project I got used to LÃ–VE2D more and learned how shaders can be used.
